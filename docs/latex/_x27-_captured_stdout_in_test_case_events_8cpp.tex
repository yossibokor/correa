\doxysubsection{Detailed Description}
\doxylink{class_test}{Test} that the captured stdout/err in (partial) test\+Case\+Ended events is correct (e.\+g. that the partial test case event does not get accumulated output).

This is done by having a single test case that is entered multiple times through generator, and a custom capturing reporter that knows what it should expect captured from the test case.
