\doxysubsection{Detailed Description}
This is a convenience header for Catch2\textquotesingle{}s Matcher support. It includes {\bfseries{all}} of Catch2 headers related to matchers.

Generally the Catch2 users should use specific includes they need, but this header can be used instead for ease-\/of-\/experimentation, or just plain convenience, at the cost of increased compilation times.

When a new header is added to either the {\ttfamily matchers} folder, or to the corresponding internal subfolder, it should be added here.
