struct copyable\+\_\+holder\+\_\+caster$<$ type, holder\+\_\+type, SFINAE $>$}
Type caster for holder types like std\+::shared\+\_\+ptr, etc. The SFINAE hook is provided to help work around the current lack of support for smart-\/pointer interoperability. Please consider it an implementation detail that may change in the future, as formal support for smart-\/pointer interoperability is added into pybind11.
