\doxysubsection{Detailed Description}
Registers custom listener that asks for captured stdout/err

Running the binary with this listener, and asking for multiple {\itshape noncapturing} reporters (one would be sufficient, but that would also mean depending on implementation details inside Catch2\textquotesingle{}s handling of listeners), we check that the listener gets redirected stdout, even though the reporters didn\textquotesingle{}t ask for it.
